# LLM RAG with Gemini, LangChain (Deep Agent) & LangSmith (Tracing)

A simple Retrieval-Augmented Generation (RAG) project built in Google Colab using **Google Gemini**, **LangChain**, and **LangSmith**.

This notebook demonstrates how to build an AI application that retrieves relevant information from a knowledge base before generating responses, while using LangSmith to trace and debug the execution.

## Features

* RAG pipeline with LangChain using Deep Agent
* Google Gemini integration
* Semantic document retrieval
* Vector database for embeddings
* LangSmith tracing and observability
* Workflow orchestration with LangChain
* Foundations for building Deep Agent applications with multi-step reasoning and tool use

## Technologies

* Python
* Google Colab
* LangChain
* Google Gemini
* LangSmith

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/denstor/Cases.git
cd Cases/24_LLM_RAG_LangChain
```

2. Open `LLM_RAG_LangChain_Gemini_LangSmith.ipynb` in Google Colab.

3. Configure your API keys:

* `GOOGLE_API_KEY`
* `LANGSMITH_API_KEY`

4. Run the notebook cells in order.
